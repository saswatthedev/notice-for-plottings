Annotation Notes

Please follow these notes carefully when annotating images:

  1. Even if a person has a beard or mustache, don't plot it if it's not clearly visible or of good quality.
  2. If hairs are not perfectly visible, don't plot them. Don't plot images with small or unclear hairs.

  4. You can plot multiple acnes, dark spots at the same time as shown in the examples.
